# Typeform 名單匯出工具 — Railway 部署說明

## 需要準備的東西

| 項目 | 說明 |
|------|------|
| Google Service Account JSON | 需要 Sheets API + Drive API 權限 |
| Typeform Token | 從 Typeform 後台取得 |
| Google Drive 資料夾 ID | 匯出的 Sheet 要放的位置 |

---

## 步驟一：Google Cloud 設定

1. 前往 [console.cloud.google.com](https://console.cloud.google.com)，選公司專案
2. 左側「API 和服務」→「程式庫」→ 啟用：
   - **Google Sheets API**
   - **Google Drive API**
3. 左側「憑證」→「建立憑證」→「服務帳戶」
   - 名稱：`typeform-exporter`
   - 角色：**編輯者**
4. 點進剛建立的服務帳戶 → 「金鑰」→「新增金鑰」→「JSON」→ 下載

---

## 步驟二：Google Drive 資料夾共用

1. 在 Google Drive 找到目標資料夾（或新建一個）
2. 右鍵 → 共用 → 加入服務帳戶 email（格式：`xxx@xxx.iam.gserviceaccount.com`）
3. 權限選「**編輯者**」
4. 複製資料夾 URL 裡的 ID（`https://drive.google.com/drive/folders/【這段】`）

---

## 步驟三：Railway 部署

1. 前往 [railway.app](https://railway.app) → New Project → Deploy from GitHub
2. 連結這個 repo（或上傳 ZIP）
3. 進入專案 → Settings → **Environment Variables**，新增以下變數：

| 變數名稱 | 值 |
|----------|----|
| `TYPEFORM_TOKEN` | Typeform 後台的 Personal Token |
| `GOOGLE_CREDENTIALS` | 服務帳戶 JSON 的**全部內容**（單行）|
| `GDRIVE_FOLDER_ID` | Drive 資料夾 ID |
| `API_SECRET` | 自訂密碼（選填，用來保護 API）|

> **GOOGLE_CREDENTIALS 轉單行指令：**
> ```bash
> cat credentials.json | python3 -c "import sys,json; print(json.dumps(json.load(sys.stdin)))"
> ```

4. Railway 會自動偵測 `Procfile` 並部署，等待 Deploy 完成
5. 複製 Railway 給的網址（例如 `https://typeform-exporter-xxx.up.railway.app`）

---

## 步驟四：設定前端

1. 打開工具網頁
2. 點「⚙ API 設定」
3. 填入 Railway 網址
4. 如果有設定 `API_SECRET`，也填入

---

## 本機測試（不需要 Google 憑證）

```bash
cd /path/to/project
pip3 install fastapi uvicorn requests

TYPEFORM_TOKEN="your_token" python3 mock_server.py
```

另開 Terminal：
```bash
python3 -m http.server 3000
```

打開 `http://localhost:3000`
